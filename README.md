# Rubix

Python written 2x2 Rubik's Cube

Notation:
L: Left
R: Right
U: Up
D: Down
F: Front
B: Back
l: Left inversed
r: Right inversed
u: Up inversed
d: Down inversed
f: Front inversed
b: Back inversed

Functions:
- displays 2x2 Rubik's cube
- notation and button controls
- random scrambler (Only valid scramblers)
